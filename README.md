# FAST ONE GAMES

![Fast One Banner](img/hero-racing2.png)

## Overview
**FAST ONE GAMES** is a premium, high-performance B2B online casino gaming platform. Designed with an ultra-sleek, aerodynamic "Carbon Fiber & Neon Red" racing aesthetic, this platform emphasizes extreme speed, seamless user experience, and cutting-edge web design to maximize player engagement.

## Key Features
- **Dynamic Speed Lines GUI**: Features continuous left-to-right pulsing data streams (Tech Lines) for an immersive high-tech feel.
- **Race-inspired Hover Effects**: Game cards trigger aggressive zoom-in and skew animations (`transform: skewX`) mimicking an F1 car dashing out of a corner.
- **Odometer Statistics Counter**: Smooth viewport-triggered `IntersectionObserver` animations that make the site's metrics (100+ Games, 50+ Currencies) rapidly count up like a sports car speedometer.
- **F1 Track Parallax**: Subtle, low-opacity SVG vector implementations of F1 racing tracks in the background, locked with `background-attachment: fixed` for a premium subtle scrolling effect.
- **No External Frameworks**: Built purely with high-performance Vanilla HTML5, CSS3, and JavaScript.

## Installation / Usage
This is a fully static front-end web application.
1. Clone or download this repository.
2. Simply open `index.html` in any modern web browser.
3. No build steps or package managers (`npm` / `yarn`) are required.

## Technologies Used
- HTML5
- CSS3 (Custom Variables, CSS Animations, Glassmorphism, Clip-paths)
- Vanilla JavaScript (DOM Manipulation, Intersection Observer API)
